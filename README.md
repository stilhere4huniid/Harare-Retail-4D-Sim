This project simulates the retail behavior of **1,000+ AI agents** representing shoppers in Harare. It uses a **Gravity Model** combined with **Behavioral Economics** to predict market share shifts across a 4-year timeline (2026 Construction to 2029 Stabilization).

It was designed to answer strategic questions for **WestProp**:
* *"How much market share will we steal from Sam Levy's Village upon launch?"*

## ğŸ“‰ Methodology
The simulation uses a modified **Huff Gravity Model**:
$$P_{ij} = \frac{U_{ij}}{\sum U_{ik}}$$
Where Utility ($U$) is a function of **Attractiveness** (Tenant Mix) divided by **Distance Friction** (Traffic).

*For a deep dive into the math and assumptions, see [METHODOLOGY.md](METHODOLOGY.md).*
